Java API Testing Framework using Rest Assured
**Overview**
This project is a Java-based API testing framework built using the Rest Assured library.
It is designed to test RESTful APIs by validating:
HTTP requests and responses
Status codes
Response body content
Headers and response time
JSON schema and business logic using assertions

**Tech Stack**
Java (JDK 8 or higher)
Rest Assured
TestNG (or JUnit)
Maven
JSON Path
Hamcrest Assertions
Jackson / Gson (for serialization & deserialization)
